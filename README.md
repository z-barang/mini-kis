# Mini-KIS

Ein kleines Krankenhausinformationssystem als Lernprojekt, gebaut mit FHIR (R4),
synthetischen Patientendaten (Synthea) und DICOM-Bildverarbeitung (pydicom).

## Status: in progress

## Stack
- Python, FastAPI
- FHIR (fhirpy, fhir.resources)
- Eigenes FHIR-Profil (FSH / SUSHI)
- pydicom
